# IO-001 ‚Äî I/O Scheduler

## Why it matters

Scheduler choice affects queueing overhead and write latency for PostgreSQL data files.

## Detection

- Resolve PostgreSQL data mount to block device
- Read `/sys/block/<device>/queue/scheduler`

## Status logic

- `none` or `mq-deadline` -> pass
- `bfq`, `cfq`, `kyber` -> warn

## Fix

```bash
echo mq-deadline > /sys/block/sda/queue/scheduler
```

Safety: `safe-runtime`

Reference: https://www.postgresql.org/docs/current/kernel-resources.html
